# Calculadora Simples

Uma calculadora simples feita em JavaScript, HTML e CSS. Este projeto permite realizar operações básicas como adição, subtração, multiplicação e divisão.

## Funcionalidades

- Adição
- Subtração
- Multiplicação
- Divisão

## Tecnologias Usadas

- HTML
- CSS
- JavaScript
